	例如  dev 分支往  pre 分支合并
	
	1. 在pre 分支的根目录下执行  
	  git config --global merge.ours.driver true
	2. 在 pre的根目录创建 .gitattributes
	   
	.env merge=ours
	 config/* merge=ours
	.idea/*  merge=ours
	.gitignore
    
     文件直接就可以，文件夹必须带着/*  否则不生效  
    .env 是文件
     config/*  是文件夹   merge=ours  固定写好
    3. dev 分支上不进行任何操作  请先在pre 分支上建立不合并的文件并且提交到pre分支上
